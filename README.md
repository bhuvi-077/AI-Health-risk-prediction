AI Healthcare Risk Prediction System

A professional Machine Learning-powered healthcare dashboard built using Python, Streamlit, and Scikit-learn for predicting patient health risk based on medical parameters.

This project demonstrates a complete end-to-end ML workflow including:

data preprocessing
feature selection
model training
scaling
prediction
evaluation
interactive web deployment

The application provides a modern dark-themed healthcare dashboard with:

real-time health risk prediction
interactive input forms
analytics visualization
metric cards
responsive UI
prediction status images
loading animations
